Today, I’d like to show off our new documentation site dedicated to SourceHut’s (and hence sr.ht’s) GraphQL APIs: docs.sourcehut.org. Our work to move from the legacy REST API to a GraphQL API is still ongoing, and certain features we are aiming for (such as federation) are still missing. Nevertheless, it is already the best - and preferred - way to programatically interact with SourceHut instances.
